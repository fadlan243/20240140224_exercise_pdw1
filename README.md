# Explore Indonesia 🌍
**Tugas Exercise 1 - Pengembangan Desain Web**

Website landing page sederhana yang menyajikan keindahan alam Indonesia, memfokuskan pada pesona pegunungan dan pantai yang megah.

## 📋 Requirement Checklist
Berikut adalah poin-poin tugas yang telah diimplementasikan:
- [x] Minimal 1 file HTML (`index.html`)
- [x] Style menggunakan file CSS eksternal (`style.css`)
- [x] Minimal 2 gambar (Gunung & Pantai)
- [x] Minimal 2 paragraf deskripsi
- [x] Penggunaan Heading H1 & H2
- [x] Terdapat tombol (Button) interaktif
- [x] Implementasi fitur Alert (JavaScript)
- [x] Komentar pada kode untuk penjelasan teknis

## 🚀 Fitur Utama
* **Responsive Design:** Tampilan tetap rapi baik di desktop maupun perangkat mobile.
* **Modern UI:** Perpaduan warna alam yang kontras dan bersih.
* **Interactive Alert:** Memberikan umpan balik saat pengguna berinteraksi dengan elemen navigasi.

## 🛠️ Cara Menjalankan
1. Clone atau download repository ini.
2. Pastikan file `gunung.jpg` dan `pantai.jpg` berada di dalam folder `images`.
3. Buka file `index.html` menggunakan browser pilihan Anda.
